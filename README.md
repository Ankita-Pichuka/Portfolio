# 🌐 Ankita Pichuka — Portfolio Website

> Personal portfolio website built with HTML, CSS & JavaScript. Live at [ankita-pichuka.netlify.app](https://ankita-pichuka.netlify.app)

---

## ✨ Features

- 🌟 Scrollytelling hero with animated star canvas
- 💼 Flip-card work experience section
- 📁 Filterable projects grid
- 🛠 Tabbed skills section
- 📬 Contact form
- 🖱 Custom cursor & floating code icons
- 📱 Fully responsive
